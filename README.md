# PiHole config

Quick setup for PiHole via docker-compose

public
```bash
git clone https://github.com/shin10/pi-hole--compose ~/.pihole && \
cd ~/.pihole && \
make setup && \
make start
```

private
```bash
git clone git@github.com:shin10/pi-hole--compose ~/.pihole && \
cd ~/.pihole && \
make setup && \
make start
```
